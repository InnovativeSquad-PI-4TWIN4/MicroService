# 🧳 EasyTrip – Plateforme de Gestion des Voyages

## 📌 Description

EasyTrip est une plateforme web développée avec **Spring Boot** permettant de gérer les réservations de voyages. Le projet est divisé en plusieurs microservices, chacun responsable d'un domaine spécifique.

Ce dépôt contient le microservice dédié à la **gestion des agences de voyage**.

## 🏢 Gestion des Agences

Ce microservice permet de gérer toutes les opérations relatives aux agences de voyage, notamment :

- Création d’une nouvelle agence
- Modification et suppression d’agence
- Consultation de la liste des agences
- Association avec des utilisateurs ou autres entités

## 💻 Technologies utilisées

- Java 17
- Spring Boot 3
- Spring Data MongoDB
- Spring Cloud (Eureka, OpenFeign)
- Lombok
- MongoDB
- API Gateway (Spring Cloud Gateway)
- Swagger
